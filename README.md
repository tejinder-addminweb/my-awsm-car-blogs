This is a Hugo static site hosted on GitHub Pages.

## Quick Start

1. Clone this repository
2. Install Hugo: https://gohugo.io/installation/
3. Run `hugo server` to start the development server
4. Visit http://localhost:1313

## Deployment

This site is automatically deployed to GitHub Pages using GitHub Actions whenever you push to the main branch.

## Adding Content

Create new posts in the `content/posts/` directory:

```bash
hugo new posts/my-new-post.md
```
